ReVersion

ReVersion is a WPF desktop application to assist managing working copies of SVN repositories spread out over multiple SVN servers. ReVersion allows developers working on many different projects, often on multiple servers, the ability to search a single location for the repository they require with the ability to checkout a number of repositories at once to assist setting up new PC's.

.net Exception Beutifier

.net Exception Beutifier is a browser extension for beautifying the Yellow Screen Of Death (YSOD) using the awesome Bootstrap themes from Bootswatch. The .net Exception Beutifier is currently avaialble on Chrome & Opera
